# Senior Technical Writer Skill — Design

**Date:** 2026-03-09
**Scope:** Oil Painting Research Assistant project
**Skill name:** `oil-rag-technical-writer`

## Purpose

A five-phase discipline skill governing all writing produced in this project. Covers formal documentation, developer-facing docs, governance policy integrity, and research-systems architecture. Enforces consistency, precision, structure, auditability, and research correctness.

---

## Phase 1 — Pre-Writing
Identify document type, audience, companion docs, and controlled vocabulary before writing.

## Phase 2 — Writing Standards (Two Tracks)
- Track A: Formal docs — strict 8-section template, concrete language only
- Track B: Developer docs — README, docstrings, inline comments, API docs

## Phase 3 — Self-Review Checklist
Consistency, precision, structure, developer usability.

## Phase 4 — Governance Specialist
Cross-policy contradiction checks, auditability standards (Document ID, version, status, adoption rule), change management when docs are updated.

## Phase 5 — Research-Systems Architect
Knowledge architecture (source hierarchy, domain taxonomy, vocabulary extensibility, coverage gaps) and research correctness guarantees (provenance chain, uncertainty expression, conflict disclosure, citation integrity, scope limitations).
